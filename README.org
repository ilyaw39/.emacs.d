This is a simple setup for GNU Emacs focusing on scientific writing in Org Mode.

-----

* Demo picture:

[[./resources/demo.png]]

-----

* Installation (Emacs 29+):

#+BEGIN_SRC shell
git clone https://github.com/ilyaw39/.emacs.d.git ~/.emacs.d
cd ~/.emacs.d/ && chmod +x setup.sh && ./setup.sh
#+END_SRC
